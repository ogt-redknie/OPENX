# `opnex tasks flow`

Flow commands are subcommands of `opnex tasks`, not a standalone `flows` command.

```bash
opnex tasks flow list [--json]
opnex tasks flow show <lookup>
opnex tasks flow cancel <lookup>
```

For full documentation see [Task Flow](/automation/taskflow) and the [tasks CLI reference](/cli/tasks).
